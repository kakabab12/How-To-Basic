#### MJPEG 스트리밍 — 라이브러리 없이 실시간 영상

**`multipart/x-mixed-replace`**는 웹의 오래된 기능이다. 서버가 JPEG를 계속 밀어 넣으면
브라우저가 **`<img>` 태그 하나로 영상처럼 재생**한다.

```html
<img src="/video_feed">
```

WebRTC도 HLS도 필요 없다. 지연이 짧고 구현이 30줄이다. **라즈베리파이 관제에는 이게 정답**이다.

#### 역할 분리

| 라우트 | 역할 |
|---|---|
| `/` | HTML 뼈대 (한 번) |
| `/video_feed` | 영상 (계속 흐름) |
| `/data` | 숫자·로그 (브라우저가 주기적으로 폴링) |
| `/control` | 명령 (버튼 누를 때) |

**영상과 데이터를 분리**한 게 핵심이다. 영상은 무겁고 끊기면 안 되고, 데이터는 가볍고
자주 갱신된다. 한 채널로 합치면 서로를 방해한다.

#### 대시보드 (`index.html`)

- 실시간 카메라 스트리밍
- CPU/메모리/FPS 모니터링
- **START / STOP / CALIB** 제어 버튼 → `/control`로 POST
- **Chart.js** 생산량·불량률 동적 차트
- 타겟 고정 시 **빨간 십자선** 표시 (`sucess.py`에서 추가된 UI)

`sub/index.html`은 여기에 더해 **불량률 10% 초과 시 경고창**을 띄운다.

#### 원격 전송 — 엣지에서 중앙 서버로

```python
def update_system_data():
    while True:
        ...
        try:
            payload = {"cpu_temp": cpu, "memory_usage": mem}
            requests.post(f"{NOTEBOOK_SERVER_URL}/api/log/system", json=payload, timeout=0.5)
        except: pass
        time.sleep(1)
```

그리고 탐지 시점에도:

```python
try:
    detection_payload = {"box_coordinates": {"x":tx,"y":ty,"z":tz},
                         "class_name": t_class, "confidence": ...}
    requests.post(f"{NOTEBOOK_SERVER_URL}/api/log/detection", json=detection_payload, timeout=0.01)
except: pass
```

**`timeout=0.01`(10ms)과 `except: pass`가 핵심**이다. 노트북 서버가 죽어 있어도
**추론 루프는 절대 멈추면 안 된다.** 로그 전송은 실패해도 그만인 부가 기능이므로
극단적으로 짧은 타임아웃 + 조용한 실패로 처리했다.

> 다만 `except: pass`(bare except)는 `KeyboardInterrupt`까지 삼킨다. 실무에서는
> `except requests.RequestException: pass`가 더 안전하다.

#### 아두이노 연결 — 자가 치유 루프

```python
def read_from_arduino():
    global ser
    while True:
        try:
            if ser is None:
                ser = serial.Serial('/dev/ttyACM0', 9600, timeout=1)
                ser.flush()
                print("✅ 아두이노 연결 성공!")
            if ser.in_waiting > 0:
                ser.readline()
        except:
            ser = None          # ★ 실패하면 None으로 되돌려 다음 루프에서 재시도
            time.sleep(5)
        time.sleep(1)
```

**USB가 뽑혔다 꽂혀도 자동 복구된다.** 예외가 나면 `ser = None`으로 리셋 → 다음 루프가
다시 연결을 시도한다. 무인 운전 장비에 필요한 패턴이다.
