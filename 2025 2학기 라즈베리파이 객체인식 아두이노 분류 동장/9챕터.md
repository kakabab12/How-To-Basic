# 9챕터 — 판정 ① 크기 — 마스크 면적

> **라즈베리파이 객체인식 + 아두이노 분류** · 9/15챕터 · [📚 목차](README.md)

---

```python
for mask, box in zip(masks, boxes):
    if len(mask) == 0: continue

    points = np.int32([mask])                    # 마스크 폴리곤 좌표
    cls_id = int(box.cls[0])
    conf = float(box.conf[0])
    original_class_name = model.names[cls_id]

    poly_area = cv2.contourArea(points)          # ★ 마스크 실제 면적
    x1, y1, x2, y2 = map(int, box.xyxy[0])
    box_area = (x2 - x1) * (y2 - y1)             # 사각형 박스 면적

    # 1. [크기 분류]
    size_prefix = "medium"
    if   poly_area < AREA_LIMIT_SMALL: size_prefix = "small"   # < 52,100
    elif poly_area > AREA_LIMIT_BIG:   size_prefix = "big"     # > 260,000
```

**`cv2.contourArea`는 폴리곤의 실제 넓이**를 구한다(신발끈 공식). 세그멘테이션 마스크가
있어야만 가능한 계산이다.

임계값 두 개(`52100`, `260000`)는 **픽셀 단위**다. 즉 **카메라 높이와 해상도에 종속**된다.
카메라를 옮기면 다시 재야 한다. 그래서 상수를 파일 맨 위에 "튜닝 포인트"로 빼 둔 것이다.

**기본값이 `medium`**인 것도 의도적이다 — 두 임계값 사이는 전부 중간 크기.
