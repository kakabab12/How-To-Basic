# 1챕터 — 무엇을 만든 것인가

> **라즈베리파이 객체인식 + 아두이노 분류** · 1/15챕터 · [📚 목차](README.md)

---

컨베이어 위 박스를 카메라가 본다 → **크기 3종 × 상태 2종 = 6가지**로 분류한다 →
아두이노에 `SORT:1`~`SORT:6`을 보내 로봇이 해당 칸으로 밀어낸다.

동시에 Flask 웹 대시보드가 **실시간 영상 + CPU/메모리/FPS + 생산량·불량률 차트**를 띄운다.

```
컨베이어 박스
    │
    ▼  Picamera2 (FHD, 오토포커스)
  프레임
    │  YOLOv8-seg (ONNX, CPU)
    ▼
  마스크 + 박스
    │  ┌─ 면적 → 대/중/소        (9단계)
    │  └─ R값  → 정상/불량       (10단계)
    ▼
 final_class_name = "big_normal_box" 같은 조합
    │  7프레임 투표               (12단계)
    ▼
 타겟 확정 → SORT:1~6 → 아두이노  (13단계)
    │
    └─→ Flask 대시보드 + 노트북 서버로 POST (14단계)
```

**핵심 설계 사상**: *"YOLO가 뱉은 클래스를 그대로 믿지 않는다."*
모델도 6클래스를 예측하지만, 코드는 **기하학적으로 다시 계산**해서 덮어쓴다.
왜 그런지가 5·9·10단계의 내용이다.
