# 10챕터 — 판정 ② 불량 — 직사각형도 R값 ★

> **라즈베리파이 객체인식 + 아두이노 분류** · 10/15챕터 · [📚 목차](README.md)

---

이 프로젝트에서 **가장 독창적인 부분**이다.

```python
rectangularity = poly_area / box_area if box_area > 0 else 0

is_abnormal = False
if "abnormal" in original_class_name:          # (1) YOLO 의견
    is_abnormal = True
if rectangularity < RECTANGULARITY_THRESHOLD:  # (2) 기하 규칙 — 0.88
    is_abnormal = True

state_suffix = "abnormal_box" if is_abnormal else "normal_box"
final_class_name = f"{size_prefix}_{state_suffix}"
```

#### R값이 무엇인가

**R = 마스크 면적 ÷ 바운딩 박스 면적**

- **완벽한 직사각형 박스**가 카메라와 정렬되어 있으면 마스크가 박스를 꽉 채운다 → **R ≈ 1.0**
- **모서리가 찌그러졌거나, 찢어졌거나, 비스듬히 놓였으면** 마스크가 박스 안의 일부만
  차지한다 → **R < 1.0**

```
정상 박스              찌그러진 박스
┌────────┐            ┌────────┐
│████████│            │███▄▄   │
│████████│  R≈0.98    │██████▄ │  R≈0.72
│████████│            │████████│
└────────┘            └────────┘
 마스크가 꽉 참         빈 공간이 많음
```

**임계값 0.88**은 "12% 이상 비면 불량"이라는 뜻이다.

#### 왜 이 방법이 좋은가

| 장점 | 설명 |
|---|---|
| **학습 불필요** | 나눗셈 한 번. 불량 샘플을 모을 필요가 없다 |
| **설명 가능** | "R=0.72라서 불량"이라고 근거를 댈 수 있다. 딥러닝은 못 한다 |
| **불량 종류 무관** | 찢김·찌그러짐·기울어짐을 **한 가지 규칙**으로 잡는다 |

**불량 데이터를 모으기 어렵다**는 게 제조 AI의 고질적 문제다. 정상품은 수천 개인데
불량품은 몇 개 없다. 이 코드는 **불량을 학습하는 대신 "정상의 정의(직사각형)"에서
벗어난 정도**를 재서 우회했다.

화면 라벨에 R값을 그대로 찍는 것도 좋은 선택이다 — 현장에서 임계값을 조정하려면
실제 값이 보여야 한다:

```python
label = f"{final_class_name} A:{int(poly_area)} R:{rectangularity:.2f}"
```
