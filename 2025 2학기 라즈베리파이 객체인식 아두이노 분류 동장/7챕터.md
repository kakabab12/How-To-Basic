# 7챕터 — 카메라 층 — Picamera2

> **라즈베리파이 객체인식 + 아두이노 분류** · 7/15챕터 · [📚 목차](README.md)

---

`camera.py` — **라즈베리파이 전용**이다. 일반 PC에서는 임포트조차 안 된다.

```python
from picamera2 import Picamera2
from libcamera import controls

class Camera:
    def __init__(self):
        try:
            self.picam2 = Picamera2()
            config = self.picam2.create_preview_configuration(
                main={"format": 'XRGB8888', "size": (1920, 1080)}      # FHD
            )
            self.picam2.configure(config)

            self.picam2.set_controls({
                "AwbMode": controls.AwbModeEnum.Auto,        # 화이트밸런스 자동
                "AfMode": controls.AfModeEnum.Continuous,    # 연속 오토포커스
                "FrameRate": 30
            })
            self.picam2.start()

            self.fps = 0
            self._prev_time = time.time()
            self.frame_count = 0

            time.sleep(2.0)      # 카메라 워밍업
            print("✅ 카메라 초기화 성공 (1920x1080 @ 30FPS, AF On)")
        except Exception as e:
            print(f"❌ 카메라 초기화 실패: {e}")
            self.picam2 = None
```

읽어낼 포인트:

- **`Picamera2`는 OpenCV `VideoCapture`가 아니다.** 라즈베리파이 카메라 모듈은 전용
  라이브러리로 제어해야 AF/AWB 같은 하드웨어 기능을 쓸 수 있다.
- **연속 오토포커스(`AfMode.Continuous`)** — 컨베이어 위 박스 높이가 달라도 초점이 따라간다.
- **`time.sleep(2.0)` 워밍업** — 카메라 센서가 안정화되기 전 프레임은 노출·화이트밸런스가
  엉망이다. 2초 기다린다.
- **실패해도 예외를 던지지 않고 `self.picam2 = None`** — 서버는 뜨게 하고, 추론 루프가
  `if model is None or cam is None: continue`로 흘려보낸다.
- **주석에 대안이 적혀 있다** — `속도가 느리다면 (640, 480)으로 변경하세요.`
  FHD는 라즈베리파이 CPU에 부담이라 실제로 조정이 필요했던 흔적이다.

`sub/camera.py`는 다른 선택을 했다 — **640x480 + 좌우 반전 + FPS를 프레임에 직접 각인**.
테스트용이라 가볍게 갔다.
