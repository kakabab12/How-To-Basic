물류 자동화 라인이다. **저울 위 화면의 무게 숫자를 읽고**, **박스 종류를 분류해서**,
기준을 넘으면 **로봇팔이 집어 옮긴다.**

```
  CAM1 (Intel RealSense)          CAM2 (USB 웹캠, 다이소)
  저울 화면을 본다                  컨베이어 박스를 본다
       │                                │
       ▼ number.engine (TensorRT)       ▼ box.pt
   'screen' 검출 → 크롭 → 640 확대       big_box / small_box
       │  (2단계 추론)                   │
       ▼ 숫자 검출                      ▼
   top-3 → x정렬 → "128"               SORT:n
       │                                │
       ▼ 98g 초과?                       ▼
    기록 (르로봇 연동 대기)          아두이노 /dev/ttyACM0
       │                                │
       └────────┬───────────────────────┘
                ▼
      FastAPI :5000 + 듀얼 스트리밍 + Claude 챗봇
```
