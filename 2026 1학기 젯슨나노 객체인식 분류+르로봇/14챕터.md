#### Pydantic 모델 — FastAPI를 쓴 이유

```python
class RobotRequest(BaseModel):
    target_item: str
    action: str

class ConveyorRequest(BaseModel):
    command: str

class ClaudeRequest(BaseModel):
    prompt: str
    context: dict
```

Flask는 `request.json.get('command')`로 **직접 꺼내고 직접 검증**해야 한다.
FastAPI는 **타입을 선언하면 자동 검증 + 자동 문서화(`/docs`)**를 해 준다.
엔드포인트가 늘어날수록 이득이 커진다.

#### Claude 챗봇 진단 — 이 프로젝트의 개성

```python
import anthropic
CLAUDE_API_KEY = "YOUR_CLAUDE_API_KEY_HERE"
CLAUDE_MODEL   = "claude-sonnet-4-20250514"

class ClaudeRequest(BaseModel):
    prompt: str
    context: dict          # ← 현장 실시간 데이터를 함께 보낸다
```

본인 설명:

> 우측 하단의 챗봇 위젯을 통해 현장의 실시간 FPS 및 CPU 데이터를 기반으로
> AI(Claude/Gemini)에게 공정 상태를 묻고 진단받을 수 있는 차세대 관제 인터페이스

**`context: dict`가 설계의 핵심**이다. 그냥 챗봇이 아니라 **현재 FPS·CPU 수치를 프롬프트에
함께 실어 보낸다.** "지금 왜 느려요?"라고 물으면 AI가 실제 수치를 보고 답한다.

라즈베리파이 편의 `FastAPI_GPU버전-수정본/app_ck.py`는 같은 발상을 **Gemini**(`google.genai`)로
했다. 두 프로젝트에 걸친 아이디어다.

> **⚠ API 키가 플레이스홀더**(`YOUR_CLAUDE_API_KEY_HERE`)로 커밋되어 있다. **키를 코드에
> 직접 넣지 말고** 환경변수로 빼는 게 안전하다:
> ```python
> import os
> CLAUDE_API_KEY = os.environ["CLAUDE_API_KEY"]
> ```
> 실수로 진짜 키를 넣고 커밋하면 git 히스토리에 영원히 남는다.

#### `templates/index.html`의 정체

`index.html`이 두 군데 있다: `project/index.html`과 `project/templates/index.html`.

FastAPI는 `FileResponse(INDEX_HTML)`로 **절대 경로를 직접** 읽는다. `templates/`는
**Flask의 관습**(`render_template`이 자동으로 찾는 폴더)이다. 라즈베리파이 편(Flask)에서
넘어온 잔재로 보인다. `RIP/` 안에 `app_ck.py`, `camera.py`가 있는 것도 같은 이유다.
