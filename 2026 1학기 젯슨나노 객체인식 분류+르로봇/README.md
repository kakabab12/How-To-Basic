# 젯슨 나노 객체인식 분류 + 르로봇 — 코드 구조 교과서

> **위치**: `kakabab12/My_project` → `2026 1학기 젯슨나노 객체인식 분류+르로봇/`
> 파일 416개 (그중 **376개가 lerobot 라이브러리** — 본인 코드는 40개)

**한 줄 요약**: 캡스톤 최종본. 엣지 보드를 **Jetson Orin Nano**로 올리고, TensorRT로
가속하고, **듀얼 카메라**로 저울의 무게 숫자와 박스를 동시에 인식해 아두이노 로봇팔을
제어한다. 웹 대시보드에 **AI 챗봇 진단**까지 붙였다.

이 폴더의 진짜 가치는 **"CSI 카메라가 고장 나서 설계를 갈아엎은 과정"이 파일로 남아
있다**는 점이다. 같은 이름의 파일이 잔뜩 있는 게 그 흔적이다.
