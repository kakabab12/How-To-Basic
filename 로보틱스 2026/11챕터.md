# 11챕터 — 문제 9 — 제어 실습 (종합)

> **로보틱스 2026** · 11/12챕터 · [📚 목차](README.md)

---

`9_control_practice.md` (31줄) + `9-1.png` ~ `9-4.png`

**여기서 1~8이 전부 합쳐진다.** 구체적인 상황 하나를 끝까지 푼다.

#### 상황

> 로봇이 정상 주행 중(**000**, 3개 센서 모두 검은색 경로 감지)에 센서값이 **100**
> (왼쪽 센서가 흰색 감지)으로 변경됨.

#### 분석

> 센서 어레이(왼쪽, 중앙, 오른쪽) 중 **왼쪽 센서가 흰색(1)을 감지**했다는 것은
> 로봇이 경로(검은색)로부터 **오른쪽으로 이탈**했음을 의미합니다.

**이 추론이 헷갈리기 쉬운 지점**이다. 왼쪽 센서가 선을 벗어났다 = **로봇이 오른쪽으로
치우쳤다** (선이 상대적으로 왼쪽에 있다).

```
   정상 (000)                이탈 (100)
   ┌───┬───┬───┐            ┌───┬───┬───┐
   │ ■ │ ■ │ ■ │            │ □ │ ■ │ ■ │   ← 왼쪽 센서가 흰 바닥
   └───┴───┴───┘            └───┴───┴───┘
   ═══════════              ═════════        ← 검은 선
     선 위 중앙                로봇이 오른쪽으로 밀림
```

#### 제어 절차 — 4단계

```
1. 오차 감지
   왼쪽 센서에서 신호(1)가 들어오는 즉시 '오른쪽 편향' 상태로 판단

2. 제어 입력 계산 (PID 제어)
   - 왼쪽 센서의 이탈 거리를 오차값(e)으로 정의
   - PID 제어기를 통해 조향값(Steering Value)을 산출
   - 조향 방식: 왼쪽 바퀴 속도(V_L)를 증가, 오른쪽 바퀴 속도(V_R)를 감소
     → 로봇을 왼쪽으로 회전

3. 경로 복귀
   중앙 센서가 다시 검은색(0)을 감지할 때까지 유지

4. 정상 주행
   모든 센서가 다시 000이 되면 PID 값을 초기화하거나 직선 주행 속도로 복귀
```

#### 모든 문제가 여기서 만난다

| 절차 | 어느 문제에서 왔나 |
|---|---|
| IR 센서로 흰색/검은색 감지 | **문제 4** (센서, 디지털 신호) |
| 이탈 거리를 오차 e로 정의 | **문제 7** (PID의 입력) |
| PID로 조향값 산출 | **문제 7** (제어) |
| **V_L 증가 + V_R 감소** | **문제 6** (차동 구동 — 속도 차이로 회전) |
| 바퀴가 실제로 회전 | **문제 2·3** (모터·바퀴) |
| PWM으로 속도 명령 전달 | **문제 8** (MCU) |
| 전부 실행 | **문제 1** (아키텍처) |

> **4번의 "PID 값을 초기화"가 중요한 디테일**이다. 적분항(I)은 **과거 오차를 누적**하므로,
> 복귀 후에도 누적값이 남아 있으면 **멀쩡한 로봇을 계속 밀어붙인다**(적분 와인드업).
> 이걸 짚은 건 PID를 개념만 외운 게 아니라는 뜻이다.
