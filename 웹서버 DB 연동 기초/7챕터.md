#### 관계형 = 표(table)로 저장

```
guestbook 테이블
┌────┬──────────┬─────────────────────┬─────────────────────┐
│ id │ name     │ message             │ created_at          │
├────┼──────────┼─────────────────────┼─────────────────────┤
│  1 │ 이지용   │ 첫 글입니다          │ 2026-07-16 01:18:14 │
│  2 │ 방문자   │ SQLite 편하네요      │ 2026-07-16 01:18:14 │
└────┴──────────┴─────────────────────┴─────────────────────┘
   ↑     ↑
  행     열 (컬럼) — 미리 정해져 있다 (스키마)
```

**핵심 성질**: **스키마가 먼저 있다.** 아무 컬럼이나 못 넣는다.

#### 5대 관계형 DB

| | **SQLite** | **MySQL** | **PostgreSQL** | **MS SQL** | **Access** |
|---|---|---|---|---|---|
| 형태 | **파일** | 서버 | 서버 | 서버 | **파일** |
| 설치 | **필요 없음** | 필요 | 필요 | 필요 | Office |
| 동시 쓰기 | **약함** (1개) | 강함 | **매우 강함** | 강함 | 매우 약함 |
| 라이선스 | Public Domain | GPL/상용 | **BSD (자유)** | 상용 | 상용 |
| 강점 | **간편함**, 내장 | 빠름, 흔함 | **기능 풍부**, 표준 준수 | Windows 통합 | GUI 쉬움 |
| 쓰는 곳 | 앱 내장, 프로토타입 | 웹서비스 다수 | **요즘 기본값** | 기업/금융 | 소규모 사무 |

#### SQLite가 특별한 이유 — 서버가 없다

```
MySQL / PostgreSQL              SQLite
──────────────────              ──────
프로그램 → TCP → DB 서버         프로그램 → 파일 직접 읽기/쓰기
             (별도 프로세스)          (같은 프로세스 안)

설치 필요                        파이썬에 내장 (import sqlite3)
포트·계정·권한 설정               파일 경로만 있으면 끝
```

**이 실습이 SQLite를 쓰는 이유**: `pip install`도 서버 실행도 필요 없다.
**`import sqlite3` 한 줄이면 끝**이다.

> **"장난감"이 아니다.** SQLite는 **세계에서 가장 많이 배포된 DB**다 —
> 안드로이드/iOS 앱, 크롬, 파이어폭스가 전부 내부에 SQLite를 쓴다.
> **동시 쓰기가 많은 웹서비스**에만 안 맞을 뿐이다.

#### Access를 이미 써봤다

> **[ASP 프로젝트](../개인%20프로젝트/2024년%202학기ASP_DB연동/README.md)의 `.mdb`가 Access**다.
> ```asp
> DSNless="DRIVER={Microsoft Access Driver (*.mdb)}; "
> DSNless=DSNless & "DBQ=" & server.mappath("nwind.mdb")
> ```
> **Access도 SQLite처럼 "파일 DB"**다. 그래서 `server.mappath()`로 **파일 경로**를
> 넘겼던 것이다. 성격이 SQLite와 같고, **SQLite가 현대판 Access**라고 봐도 된다
> (Windows 종속이 없고 무료라는 점만 빼면).

#### SQL 방언 — 표준이 있어도 다 다르다

| 하고 싶은 것 | SQLite | MySQL | PostgreSQL |
|---|---|---|---|
| 자동증가 | `INTEGER PRIMARY KEY AUTOINCREMENT` | `INT AUTO_INCREMENT` | `SERIAL` |
| 상위 N개 | `LIMIT 10` | `LIMIT 10` | `LIMIT 10` |
| 문자열 연결 | `\|\|` | `CONCAT()` | `\|\|` |
| 현재 시각 | `CURRENT_TIMESTAMP` | `NOW()` | `NOW()` |

> **[ASP의 `DELETE * FROM`](../개인%20프로젝트/2024년%202학기ASP_DB연동/README.md)이
> Access 방언**이었던 것과 같은 이야기다. 표준 SQL은 `DELETE FROM`이다.
>
> **그래서 `schema.sql`을 따로 뺐다**(11단계). DB를 옮길 때 **이 파일만 고치면 된다.**
