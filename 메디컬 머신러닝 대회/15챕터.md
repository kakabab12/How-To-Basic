#### ✅ 서버는 CSV 없이 바로 뜬다

**학습된 `.pkl`이 전부 커밋되어 있어서** 데이터 없이 실행된다. 이 저장소에서
**가장 쉽게 돌아가는 프로젝트**다.

```bash
cd "메디컬 머신러닝 대회/AI"
pip install flask flask-cors joblib numpy pandas lightgbm xgboost scikit-learn
py server.py
# http://localhost:5000
```

```python
if __name__ == '__main__':
    print("http://localhost:5000")
    app.run(debug=True, port=5000)
```

> **`host` 지정이 없다** → 기본값 `127.0.0.1`이라 **로컬에서만** 접속된다.
> 다른 기기에서 보려면 `app.run(host='0.0.0.0', port=5000)`으로 바꿔야 한다.
> (앞의 두 프로젝트는 `host='0.0.0.0'`이었다 — 라즈베리파이/젯슨은 원격 접속이
> 필수였고, 여기는 발표용 로컬 데모라 그럴 필요가 없었다.)

#### ❌ 재학습은 데이터가 필요하다

```bash
# 1. PhysioNet에서 INSPIRE 1.3 신청·다운로드
# 2. operations.csv, diagnosis.csv, labs.csv, ward_vitals.csv 를 AI/ 폴더에
py train.py
```

**주의**: `labs.csv`, `ward_vitals.csv`는 대용량이다. `chunksize=500000`으로 나눠
읽지만(7단계) `concat` 후에는 필터링된 전체가 메모리에 올라간다. RAM이 넉넉해야 한다.
