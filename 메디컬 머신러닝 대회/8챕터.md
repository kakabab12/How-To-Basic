# 8챕터 — 학습 5/7 — 생체징후

> **메디컬 머신러닝 대회** · 8/15챕터 · [📚 목차](README.md)

---

```python
ward_pivot = ward.pivot_table(
    index='subject_id', columns='item_name',
    values='value', aggfunc=['mean', 'min', 'max', 'std', 'last']   # ★ std 추가
)
```

**7단계와 거의 같지만 `std`(표준편차)가 추가**됐다.

**왜 생체징후에만 `std`를 넣었나**: 혈압·심박수는 **변동성 자체가 위험 신호**다.
평균 심박수 80인 두 환자라도, 한 명은 78~82로 안정적이고 다른 한 명은 50~120을
오간다면 **후자가 훨씬 위험**하다. 혈액검사는 그렇게 자주 재지 않으니 `std`가 의미가 적다.

**같은 도구(`pivot_table`)를 쓰면서 데이터 성격에 따라 집계를 다르게** 한 것 —
도메인 이해가 반영된 지점이다.

`ward_vitals`(병동 생체징후)를 쓰고 `vitals`(수술 중 생체징후)를 안 쓴 것도 같은 맥락이다.
**병동 = 수술 전, 수술 중 = 미래.**
