# 1챕터 — 무엇을 만든 것인가

> **메디컬 머신러닝 대회** · 1/15챕터 · [📚 목차](README.md)

---

의사가 수술 전에 환자 정보를 입력하면:

- **"이 환자가 ICU에 갈 확률은 몇 %인가"** (분류)
- **"수술이 몇 분 걸릴까"** (회귀, 최단~최장 범위까지)
- **"왜 그렇게 판단했나"** (SHAP 근거 + 임상 소견)

를 답해 준다.

```
  의료진 (index.html)
      │  나이·ASA·응급여부·진단·혈액검사 입력
      ▼
  Flask /predict (server.py)
      │
      ├─▶ model_cls.pkl (LightGBM 분류) ──▶ ICU 확률 (원시)
      │        │
      │        ▼ clinical_calibrate()  ★ 13단계
      │     로짓 공간에서 의료 지식으로 보정
      │        │
      │        ▼ 보정된 확률 + 위험 등급
      │
      └─▶ model_reg.pkl (LightGBM 회귀) ──▶ 수술 시간 (분)
               │
               ▼ resid_std.pkl 로 최단/최장 범위
      │
      ▼
  원형 게이지 + 결정적 요인 막대 + 종합 임상 소견
```

**두 모델을 동시에** 쓰는 게 특징이다. 하나는 분류, 하나는 회귀인데 **같은 피처**를 공유한다.
