### 읽어낼 흐름

- **학교 과제(2020~2025)** → **개인 프로젝트(고클린)**로 넘어간다
- 기술이 **웹(Express/ASP/HTML) → 데스크톱 앱(Python GUI)**으로 이동
- 그리고 저장소의 나머지 폴더를 보면 **결국 AI·로보틱스로** 간다.
  **이 폴더는 그 이전의 기록**이다

> **고클린만 "과제"가 아니다.** 나머지는 전부 학교에서 시킨 것이고, 고클린은
> **본인이 필요해서 만든 것**이다. 그래서 완성도(다국어·exe 배포·후원 기능)가 다르다.

## 🔗 이 폴더가 저장소 전체와 이어지는 지점

**2024년 임베디드 프로젝트가 이후 모든 비전 프로젝트의 출발점**이다:

| 개념 | 2024 임베디드 | → | 이후 |
|---|---|---|---|
| **Picamera2** | `picam2.start()` | → | **2025 라즈베리파이**: AF·AWB 제어까지 |
| **`while True` + `waitKey('q')`** | `bodycam.py` | → | **2026 광명테크** `run_demo.py`에 그대로 |
| **`finally`로 하드웨어 정리** | `bodycam.py` | → | **2026 광명테크** `runner.stop()` |
| **센서 → 판정 → 표시** | 초음파 → `<10cm` → 화면 | → | **2025**: YOLO → R값 → `SORT:n` |
| **주야간 검증** | 영상 2개 | → | **2026**: 단위 테스트 36건 |

### 판정 로직의 진화

```python
# 2024 임베디드 — 초음파 거리
if distance < 10:
    return "Baedal object detect!!"

# 2025 라즈베리파이 — 직사각형도
rectangularity = poly_area / box_area
if rectangularity < 0.88:
    is_abnormal = True

# 2026 광명테크 — 벡터 외적
cross = v1x * v2y - v1y * v2x
is_palm_facing = cross > 0 if handedness_label == "right" else cross < 0
```

**전부 "물리량을 재서 임계값과 비교"다.** 거리 → 면적 비율 → 벡터 외적.
**학습 없이 기하로 판정한다**는 철학이 처음부터 같다.

## ⚡ 빠른 시작 — 지금 되는 것

| 프로젝트 | 명령 | 바로 되나 |
|---|---|---|
| **고클린 alpha.ver** | `py alpha_test1.py` 또는 `dist/alpha_test1.exe` | ✅ **바로 됨** (UAC 뜸) |
| **고클린 초기.ver** | `update.bat` 더블클릭 | ✅ 바로 됨 |
| **XAMPP 웹** | `Web/index.html` 브라우저로 열기 | ⚠ 이미지·영상 깨짐 |
| **Express 도서관** | `npm install && npm start` | ❌ `/` 접속 시 500 에러 |
| **ASP DB 연동** | IIS 설정 필요 | ❌ 설정 복잡 |
| **임베디드** | 라즈베리파이 + 센서 필요 | ❌ 하드웨어 |

> **`python`이 아니라 `py`를 쓰는 이유**: 이 PC에서 `python`은 Microsoft Store 스텁이라
> 아무것도 하지 않는다. 실제 파이썬은 **`py` 런처**(3.13)로 실행된다.

## 🔍 각 설명서에서 확인한 것들

**코드와 파일을 직접 열어 확인한 사실**들이다.

| 프로젝트 | 발견 |
|---|---|
| **임베디드** | **`.py` 파일이 없는데 코드는 있다** — `.hwpx`(=ZIP) 안 보고서에 전문이 들어 있다 |
| **임베디드** | `fooddetect.py`의 `get_distance()`에 **타임아웃이 없다** — 센서가 죽으면 무한 루프 |
| **임베디드** | `bodycam.py`는 `finally`로 정리하는데 **`fooddetect.py`는 안 한다** — 같은 보고서인데 품질이 다름 |
| **Express** | **`Board`가 정의되지 않았다** — `/` 접속 시 `ReferenceError`로 500 |
| **Express** | `models/` 폴더 자체가 없다 + 폴더 이름이 **`복구`** → **파일 유실 사건**의 증거 |
| **ASP** | **전 파일이 SQL 인젝션에 취약** — 문자열 연결로 SQL 생성 |
| **ASP** | `cread.asp`가 **`<meta refresh 3초>`로 실시간 게시판**을 구현 (AJAX 이전 방식) |
| **ASP** | `DELETE *` — **Access 전용 방언** (표준 SQL은 `DELETE FROM`) |
| **XAMPP** | **주석이 채점 기준표**다 (`<!-- 1. 본인 사진 -->` … `<!-- 8. 검색엔진 -->`) |
| **XAMPP** | **이미지 5개가 전부 깨진다** — `Web/`에 없고 `웹 과제/`에 있다 |
| **XAMPP** | **YouTube가 안 나온다** — `watch?v=` URL은 iframe에 못 넣는다(`embed/`여야 함) |
| **고클린 초기** | **NVIDIA가 하드코딩** — alpha.ver에서 GPU 자동 감지로 개선됨 |
| **고클린 alpha** | **게임 모드에 확인이 없다** — 초기.ver의 Y/N 안전장치가 사라짐 |
| **고클린 alpha** | `sfc /scannow`가 빠짐 — GUI에서 30분 작업의 진행률을 못 보여줘서 |
