# 4챕터 — 관리자 권한 자동 획득

> **고클린 alpha.ver** · 4/15챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

**파일 맨 위, GUI를 만들기도 전에 실행되는 코드**다.

```python
import ctypes, sys

# === 1. 관리자 권한 자동 획득 ===
def is_admin():
    try:
        return ctypes.windll.shell32.IsUserAnAdmin()
    except:
        return False

if not is_admin():
    ctypes.windll.shell32.ShellExecuteW(None, "runas", sys.executable, " ".join(sys.argv), None, 1)
    sys.exit()
```

#### 왜 관리자 권한이 필요한가

| 명령 | 왜 관리자 전용인가 |
|---|---|
| `del %TEMP%` | 시스템 폴더 파일 삭제 |
| `netsh winsock reset` | 네트워크 스택 설정 변경 |
| `dism` | 시스템 이미지 조작 |
| `defrag` | 디스크 직접 접근 |
| `taskkill` (타 사용자 프로세스) | 프로세스 강제 종료 |

**권한 없이 실행하면 전부 "액세스가 거부되었습니다"**로 끝난다. 그런데 GUI 프로그램은
그 에러를 사용자에게 잘 보여주지도 못한다. **애초에 권한을 받고 시작하는 게 맞다.**

#### 흐름

```
사용자가 실행
    │
    ▼
is_admin() ?
    │
    ├── True  → 계속 진행 (GUI 생성)
    │
    └── False → ShellExecuteW("runas", ...)  ← UAC 창을 띄우며 새 프로세스 시작
                    │
                    └── sys.exit()           ← 현재(권한 없는) 프로세스는 즉시 죽는다
                          │
                          ▼
                    새 프로세스가 처음부터 다시 → is_admin() == True → GUI
```

**`sys.exit()`가 핵심이다.** 없으면 권한 없는 창과 권한 있는 창이 **두 개 뜬다.**

#### 세부

| 코드 | 의미 |
|---|---|
| `ctypes.windll.shell32` | **Windows API를 직접 호출**. `shell32.dll` 로드 |
| `IsUserAnAdmin()` | 정식 Windows API — 관리자 여부 반환 |
| `ShellExecuteW(...)` | `W` = 유니코드 버전 (한글 경로 대응) |
| **`"runas"`** | **이 동사가 UAC 창을 띄운다** |
| `sys.executable` | 현재 파이썬 인터프리터 경로 (`.exe`로 빌드하면 그 exe) |
| `" ".join(sys.argv)` | 원래 인자를 그대로 전달 |
| 마지막 `1` | `SW_SHOWNORMAL` — 창을 정상 크기로 |

#### 배치 버전과 비교

| | 초기.ver (배치) | alpha.ver (Python) |
|---|---|---|
| 권한 확인 | **`fltmc` 실패 여부** (간접 추론) | **`IsUserAnAdmin()`** (정식 API) |
| 재실행 | `powershell Start-Process -Verb RunAs` | `ShellExecuteW(..., "runas", ...)` |

배치는 **관리자만 되는 명령을 일부러 실행해 보고 실패하면** 권한이 없다고 판단한다
(`fltmc` 트릭). Python은 `ctypes`로 **API를 직접 물어본다.** 같은 목적, 다른 수단이다.

> **`try/except`로 감싼 이유**: `IsUserAnAdmin`은 아주 오래된 Windows나 비정상 환경에서
> 실패할 수 있다. 그때는 `False`(권한 없음)로 간주해 UAC를 띄운다 — **안전한 쪽으로 실패**한다.
