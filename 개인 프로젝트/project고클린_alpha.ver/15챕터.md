# 15챕터 — PyInstaller 배포와 실행법

> **고클린 alpha.ver** · 15/15챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

#### 왜 `.exe`로 만드나

**이 프로그램의 사용자는 개발자가 아니다.**

```
❌ "Python 3.11 설치하고, pip install customtkinter 하고, 
    관리자 권한으로 py alpha_test1.py 실행하세요"

✅ alpha_test1.exe 더블클릭
```

**PC 최적화 도구를 쓰려는 사람에게 파이썬 설치를 요구할 수 없다.**

#### 빌드 산출물

| 파일/폴더 | 정체 |
|---|---|
| **`alpha_test1.spec`** | **빌드 레시피** — `--onefile`, 아이콘, 숨은 임포트 등 |
| `build/` | **중간 산출물** — 보통 `.gitignore` 대상 |
| `build/Analysis-00.toc` | 의존성 분석 결과 (어떤 모듈을 넣을지) |
| `build/PYZ-00.pyz` | 압축된 파이썬 모듈 묶음 |
| **`build/warn-alpha_test1.txt`** | **빌드 경고** — 문제 생기면 여기부터 본다 |
| `build/xref-alpha_test1.html` | 임포트 상호참조 (누가 뭘 부르는지) |
| **`dist/alpha_test1.exe`** | **최종 결과물** |
| `alpha_test1.exe` | 루트 복사본 |

#### 빌드 명령

```bash
pyinstaller alpha_test1.spec
# 또는 처음부터
pyinstaller --onefile --windowed --uac-admin alpha_test1.py
```

| 옵션 | 의미 |
|---|---|
| `--onefile` | 단일 exe로 묶기 |
| `--windowed` | **콘솔 창 없이** (GUI 앱) |
| `--uac-admin` | 실행 시 UAC 요청 (4단계를 매니페스트로 대체 가능) |

#### 실행

```bash
cd "개인 프로젝트/project고클린_alpha.ver"
pip install customtkinter
py alpha_test1.py
```

또는 빌드된 것을 바로:

```
dist/alpha_test1.exe   (또는 alpha_test1.exe)
```

**실행하면 UAC(사용자 계정 컨트롤) 창이 뜬다. 정상이다** (4단계).

> **`python`이 아니라 `py`를 쓰는 이유**: 이 PC에서 `python`은 Microsoft Store 스텁이라
> 아무것도 하지 않는다. 실제 파이썬은 **`py` 런처**(3.13)로 실행된다.

#### ⚠ 실행 전 알아 둘 것

| # | 주의 |
|---|---|
| 1 | **실제로 시스템을 건드린다.** Temp 삭제, `winsock reset`, 프로그램 강제 종료 |
| 2 | **게임 모드는 확인 없이 바로 종료한다** (13단계) — 작업 중인 문서를 먼저 저장할 것 |
| 3 | **`netsh winsock reset` 후 재부팅**이 필요할 수 있다 |
| 4 | **백신 오탐 가능성** — PyInstaller exe + 관리자 권한 + `taskkill`/`netsh` 조합은 백신이 싫어한다 |
| 5 | **Windows 전용** — `wmic`·`winget`·`defrag`·`STARTUPINFO` 전부 Windows API |
| 6 | **`wmic`은 Windows 11에서 deprecated** — 없으면 PC 정보 일부가 빈다 (10단계의 `except: return ""` 덕에 죽지는 않는다) |

#### `build/`와 `dist/`가 커밋된 것에 대해

**보통은 `.gitignore`에 넣는다** — 소스에서 재생성되니까.

**다만 장단이 있다:**

| | |
|---|---|
| ✅ | 저장소를 받은 사람이 **빌드 없이 바로 실행** |
| ❌ | 저장소가 무거워짐, 소스와 exe가 **어긋날 수 있음** |

이 프로젝트는 **배포가 목적**이라 전자를 택한 것으로 보인다.

#### 이 프로그램에서 배울 점

| 습관 | 어디서 |
|---|---|
| **새로 만들지 않는다** — 있는 걸 조합한다 | 1단계 (전부 Windows 내장 명령) |
| **권한은 먼저 받는다** | 4단계 |

| **할 수 있는 것/없는 것을 구분한다** | 12단계 (드라이버는 페이지까지만) |
| **사용자는 개발자가 아니다** | 11단계(영어 기본), 14단계(계좌 복사), 15단계(exe) |
