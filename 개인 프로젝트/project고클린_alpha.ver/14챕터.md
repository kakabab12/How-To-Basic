#### ⑤ `start_info` — 명령 4종의 출력을 한 화면에

| 정보 | 명령 |
|---|---|
| CPU | `wmic cpu get name` |
| 메인보드 | `wmic baseboard get product,manufacturer` |
| GPU | `wmic path win32_VideoController get name` |
| RAM | `powershell` (바이트 → GB 계산) |
| 내부 IP | `ipconfig` |
| 외부 IP | `curl ifconfig.me` 등 |

**RAM만 `powershell`을 쓰는 이유**: `wmic`은 총 메모리를 **바이트**로 준다
(`17179869184`). GB로 바꾸려면 계산이 필요한데, `powershell`은 한 줄로 된다:

```powershell
(Get-CimInstance Win32_ComputerSystem).TotalPhysicalMemory / 1GB
```

**도구를 목적에 맞게 골라 썼다.**

**내부 IP와 외부 IP를 둘 다** 보여주는 것도 실용적이다 — 공유기 뒤에 있으면 두 값이
다르고, 그 차이 자체가 네트워크 구성을 말해 준다.

#### `start_about` — 75줄짜리 후원 팝업

본인 설명:

> `ctk.CTkToplevel`을 사용해 메인 창 위에 작은 팝업창을 띄우고, 후원 링크를 웹브라우저로
> 열거나(`webbrowser.open`), 카카오뱅크 계좌번호를 클립보드에 바로 복사해 주는
> (`clipboard_append`) 디테일한 편의 기능을 구현했습니다.

| 요소 | 역할 |
|---|---|
| `ctk.CTkToplevel` | **별도 창**. 메인 창 위에 뜬다 |
| `webbrowser.open(url)` | 후원 링크를 기본 브라우저로 |
| `clipboard_append(계좌번호)` | **클립보드에 자동 복사** |

**계좌번호 클립보드 복사가 좋은 UX다.**

```
❌ "카카오뱅크 3333-01-1234567 로 보내주세요"   → 사용자가 받아 적거나 드래그해야 함
✅ [계좌번호 복사] 버튼                          → 클릭 한 번, 바로 붙여넣기
```

계좌번호는 **길고 하이픈이 섞여 있어 옮겨 적다 틀리기 쉽다.** 한 번의 클릭으로 없앴다.

#### 75줄이나 되는 이유

**기능(`start_ping` 35줄, `start_gamemode` 36줄)보다 크다.** 팝업 창을 새로 만들려면:

```
창 생성 → 크기·위치 설정 → 모달 설정 → 라벨 여러 개 → 버튼 여러 개
→ 각 버튼 콜백 → 레이아웃 배치 → 닫기 처리
```

**GUI 코드는 원래 장황하다.** 로직 3줄을 위해 위젯 코드 20줄이 붙는다.
7단계의 스레드 패턴이 5번 반복돼도 부담이 적었던 것과 대조된다.

> **이 프로그램이 "개인 프로젝트"인 이유가 여기 있다.** 학교 과제에는 후원 버튼을 안 넣는다.
> 배포해서 남이 쓸 것을 전제한 기능이고, 11단계의 "기본 언어 영어"와 15단계의 `.exe` 배포가
> 같은 방향을 가리킨다.
