#### ① `start_update` — GPU 제조사 자동 감지

본인 설명:

> `wmic` 명령어로 사용자의 그래픽카드 제조사(NVIDIA, AMD, Intel)를 자동 감지하여, 해당
> 제조사의 공식 드라이버 다운로드 웹사이트를 브라우저로 열어주는 똑똑한 기능이 포함되어
> 있습니다.

하는 일:
- `winget upgrade` — 설치된 프로그램 업데이트
- `pip` 업데이트 — 파이썬 패키지
- Windows 업데이트 / MS Store 창 자동 실행
- **GPU 감지 → 드라이버 페이지 열기**

```
wmic path win32_VideoController get name
        ↓
"NVIDIA GeForce RTX 3060" 같은 문자열
        ↓
"NVIDIA" in name ?  →  webbrowser.open("https://www.nvidia.com/...")
"AMD"    in name ?  →  AMD 드라이버 페이지
"Intel"  in name ?  →  Intel 드라이버 페이지
```

#### 왜 "열어 주기"에서 멈추나

**드라이버는 자동 설치가 불가능하다.**

| 이유 | 설명 |
|---|---|
| 제조사마다 다름 | NVIDIA/AMD/Intel 설치 방식이 전부 다르다 |
| 라이선스 동의 필요 | 사용자가 직접 눌러야 한다 |
| GPU 모델별 파일 다름 | 정확한 모델을 알아야 한다 |
| 잘못 설치하면 부팅 불가 | 위험이 크다 |

**할 수 있는 것과 없는 것을 구분한 판단**이다. "자동으로 다 해 줌"을 약속하고 망가뜨리는
것보다, **정확한 페이지까지 데려다주고 손을 떼는** 게 옳다.

> 이 태도는 8단계의 `indeterminate`(모르면 모른다고 함)와 같은 결이다.

#### ② `start_clean` — Windows 내장 명령 총동원

| 명령 | 하는 일 |
|---|---|
| `ipconfig /flushdns` | DNS 캐시 초기화 — 접속 안 되는 사이트 문제 해결 |
| `del %TEMP%\*` | 임시 파일 강제 삭제 — 디스크 확보 |
| `dism /online /cleanup-image` | 시스템 이미지 정리 — 오래된 업데이트 파일 제거 |
| `netsh winsock reset` | 소켓 카탈로그 초기화 — 네트워크 이상 복구 |
| `defrag` | 디스크 조각 모음 |

**전부 Windows 내장**이라 설치할 것도, 의존성도 없다.

> **초기.ver에는 `sfc /scannow`(시스템 파일 무결성 검사)도 있었는데 alpha.ver에서 빠졌다.**
> `sfc`는 10~30분이 걸린다 — GUI에서 그렇게 오래 도는 건 부담스러웠을 것이다.
>
> **`netsh winsock reset`은 재부팅이 필요하다.** 실행 후 알려 주는 게 좋다.
