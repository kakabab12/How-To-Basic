#### 568줄의 구성

```
줄       내용
────────────────────────────────────────────
1~8      import
10~19    === 1. 관리자 권한 자동 획득 ===     ← 4단계
21~31    === 2. 테마 설정 ===                ← 5단계
33~175   class SystemMasterApp.__init__      ← 5·6·11단계 (GUI 조립 + 다국어 딕셔너리)
176~192  toggle_language                     ← 11단계
193~221  log_summary / safe_decode /
         get_cmd_output / log                ← 9·10단계
222~250  set_working_state                   ← 8단계
252~326  start_about (후원 팝업)             ← 14단계  (75줄!)
328~409  start_update                        ← 12단계
411~461  start_clean                         ← 12단계
463~498  start_gamemode                      ← 13단계
500~534  start_ping                          ← 13단계
536~568  start_info                          ← 14단계
```

**`__init__`이 142줄로 가장 길다.** GUI 위젯을 전부 여기서 만들고 다국어 딕셔너리도
여기 들어 있다. 파일을 나누거나 위젯 생성을 메서드로 쪼갤 여지가 있지만,
**568줄짜리 단일 파일 도구로는 충분히 읽힌다.**

> **`start_about`이 75줄**로 실제 기능(`start_ping` 35줄, `start_gamemode` 36줄)보다 크다.
> 후원 팝업에 그만큼 공을 들였다는 뜻이다(14단계).
