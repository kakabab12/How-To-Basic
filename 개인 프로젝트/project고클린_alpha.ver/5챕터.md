```python
import customtkinter as ctk

# === 2. 테마 설정 ===
ctk.set_appearance_mode("Dark")
ctk.set_default_color_theme("dark-blue")

# 색상 정의
COLOR_BG           = "#0f0f0f"    # 거의 검정
COLOR_SIDEBAR      = "#181818"    # 살짝 밝은 검정
COLOR_ACCENT       = "#00E5FF"    # 시안
COLOR_BTN_HOVER    = "#00B8D4"    # 진한 시안
COLOR_TERMINAL_BG  = "#000000"    # 순수 검정
COLOR_TERMINAL_TXT = "#00FF41"    # ★ 매트릭스 그린
```

#### `customtkinter`를 쓴 이유

**기본 `tkinter`는 1990년대 룩**이다 — 회색 배경, 각진 버튼, 호버 효과 없음.
`customtkinter`는 그 위에 씌운 현대적 래퍼로 **다크모드·둥근 모서리·호버**를 준다.

| | tkinter | customtkinter |
|---|---|---|
| 외형 | Windows 95 | 현대적 다크/라이트 |
| 의존성 | **파이썬 기본 내장** | `pip install customtkinter` |
| API | `tk.Button` | `ctk.CTkButton` (거의 동일) |

**API가 거의 같아서 배우는 비용이 작다.** 이 프로그램의 **유일한 외부 의존성**이기도 하다.

#### 색상을 상수로 뽑은 이유

**테마를 바꾸려면 6줄만 고치면 된다.** 위젯마다 `"#00E5FF"`를 하드코딩했다면
수십 군데를 찾아 고쳐야 한다.

#### 색이 말하는 것

| 색 | 어디 | 의미 |
|---|---|---|
| `#00FF41` **매트릭스 그린** | 터미널 글씨 | "해커/전문가 도구" |
| `#00E5FF` **시안** | 강조·완료 | 기술적/차가운 느낌 |
| `#FFD700` **금색** | 작업 중 상태 | 경고/진행 (8단계) |
| `#0f0f0f` / `#000000` | 배경 | 다크 |

**초기.ver 배치 버전의 `color 0b`(검정 바탕 + 하늘색)에서 이어지는 미학**이다.
시스템 도구는 "전문가용"으로 보여야 신뢰가 간다는 판단이 두 버전 모두에 깔려 있다.
