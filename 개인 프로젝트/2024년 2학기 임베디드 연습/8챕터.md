#### 실행 (라즈베리파이가 있다면)

**코드를 보고서에서 꺼내 `.py`로 저장하면 된다:**

```bash
# 라즈베리파이에서
sudo python3 bodycam.py       # 바디캠
sudo python3 fooddetect.py    # 배달 감지
```

**필요한 것:**

| 항목 | 내용 |
|---|---|
| 보드 | 라즈베리파이 (GPIO + 카메라 슬롯) |
| 카메라 | 라즈베리파이 카메라 모듈 |
| 센서 | **HC-SR04 초음파** (배달 감지용) |
| 부품 | LED + 저항 |
| 라이브러리 | `picamera2`, `RPi.GPIO`, `opencv-python` |
| 권한 | **`sudo`** (GPIO 접근) |

**배선:**

```
bodycam.py                fooddetect.py
  GPIO 17 ── LED            GPIO 23 ── TRIG (HC-SR04)
                            GPIO 24 ── ECHO (HC-SR04)  ※ 분압 저항 필요!
                            GPIO 25 ── LED
```

> **⚠ HC-SR04의 ECHO는 5V를 출력한다.** 라즈베리파이 GPIO는 **3.3V 전용**이라
> **직결하면 핀이 손상될 수 있다.** 분압 저항(1kΩ + 2kΩ)이 필요하다.
> 보고서에는 이 언급이 없다 — 실습에서 넘어갔거나 모듈에 이미 있었을 수 있다.

#### 이 프로젝트가 시작한 것들

**2024년 이 폴더에서 시작해서 2026년까지 이어지는 것:**

| 개념 | 2024 임베디드 | → | 이후 |
|---|---|---|---|
| **Picamera2** | `picam2.start()` | → | **2025 RPi**: AF·AWB 제어까지 |
| **`while True` + `waitKey('q')`** | 여기서 시작 | → | **2026 광명테크** `run_demo.py`에 그대로 |
| **`finally` 로 하드웨어 정리** | `bodycam.py` | → | **2026 광명테크** `runner.stop()` |
| **`cv2.putText` 로 상태 표시** | `status` 오버레이 | → | **2025 RPi**: `A:{면적} R:{R값}` 라벨 |
| **센서 → 판정 → 표시** | 초음파 → `<10` → 화면 | → | **2025 RPi**: YOLO → R값 → SORT |
| **주야간 검증** | 영상 2개 | → | **2026**: 단위 테스트 36건 |
| **실제 불편에서 문제 정의** | "배달을 두고 간다" | → | **로보틱스**: 요구사항 5개 |

#### 판정 로직의 진화

**이 프로젝트의 판정:**
```python
if distance < 10:
    return "Baedal object detect!!"
```

**1년 뒤 (2025 라즈베리파이):**
```python
rectangularity = poly_area / box_area
if rectangularity < RECTANGULARITY_THRESHOLD:      # 0.88
    is_abnormal = True
```

**2년 뒤 (2026 광명테크):**
```python
cross = v1x * v2y - v1y * v2x
is_palm_facing = cross > 0 if handedness_label == "right" else cross < 0
```

**전부 "물리량을 재서 임계값과 비교"다.**
거리 → 면적 비율 → 벡터 외적. **학습 없이 기하로 판정한다**는 철학이 처음부터 같다.

> **[2025 라즈베리파이 10단계](../../2025%202학기%20라즈베리파이%20객체인식%20아두이노%20분류%20동장/README.md)에서
> "YOLO를 믿지 않고 R값으로 재계산"한 것**과, **[광명테크 7단계](../../광명테크/README.md)에서
> "학습 대신 외적 부호"**를 택한 것 — **뿌리가 이 초음파 센서**에 있다.
> **"측정할 수 있으면 측정한다"**가 일관된 태도다.
