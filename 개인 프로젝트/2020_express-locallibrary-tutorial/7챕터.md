# 7챕터 — ★ `routes/index.js` — 그리고 `Board` 버그

> **Express 지역 도서관 (2020)** · 7/10챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

**파일 전체가 12줄**이다:

```javascript
var express = require('express');
var router = express.Router();

/* GET home page. */
router.get('/', function(req, res, next) {
  Board.find({}, function (err, board) {                        // ★ 문제의 줄
      res.render('index', { title: 'Express', board: board });
  });
});

module.exports = router;
```

#### `Board`가 정의되어 있지 않다

**`require` 목록을 보자:**

```javascript
var express = require('express');    // ← express만 가져온다
var router = express.Router();
```

**`Board`를 가져오는 줄이 없다.** 파일 어디에도 `Board`의 정의가 없다.
`models/` 폴더도 없다(3단계).

#### 실행하면 무슨 일이 일어나나

```
브라우저: http://localhost:3000/  접속
    ↓
router.get('/') 핸들러 실행
    ↓
Board.find({}, ...)
    ↓
ReferenceError: Board is not defined
    ↓
Express 에러 핸들러(app.js ⑧) → 500 Internal Server Error
```

**홈페이지가 열리지 않는다.** 이 앱의 유일한 페이지가 `/`인데 그게 죽는다.

> **즉 이 프로젝트는 현재 상태로는 동작하지 않는다.** 3단계의 "복구" 이야기가
> 여기서 확정된다 — **`models/Board.js`가 사라졌고, 그걸 쓰는 코드만 남았다.**

#### 원래 어떤 코드였을까

```javascript
// 사라진 models/Board.js (추정)
var mongoose = require('mongoose');
var BoardSchema = new mongoose.Schema({
    title: String,
    content: String,
    // ...
});
module.exports = mongoose.model('Board', BoardSchema);

// routes/index.js 에 있었어야 할 줄
var Board = require('../models/Board');    // ← 이게 없다
```

#### 살리려면

```javascript
// routes/index.js 최소 수정 — DB 없이 페이지만 띄우기
router.get('/', function(req, res, next) {
  res.render('index', { title: 'Express' });
});
```

**`Board`를 안 쓰면 바로 돌아간다.** `views/index.pug`가 `board`를 쓰지도 않기 때문이다(8단계).

#### 콜백 스타일이 시대를 말한다

```javascript
Board.find({}, function (err, board) {     // ← 콜백
    res.render('index', { ... });
});
```

**2020년 스타일**이다. 지금은 이렇게 쓴다:

```javascript
router.get('/', async (req, res) => {
  const board = await Board.find({});      // ← async/await
  res.render('index', { title: 'Express', board });
});
```

**`err`를 확인조차 안 한다**는 것도 문제다. DB 에러가 나도 그냥 `board`가 `undefined`인
채로 렌더링된다.

#### `routes/users.js`
