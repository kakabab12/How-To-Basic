# 1챕터 — 정체 — MDN 튜토리얼

> **Express 지역 도서관 (2020)** · 1/10챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

**MDN Web Docs의 Express 튜토리얼**을 따라 한 것이다. 폴더 이름이 그대로 말해 준다:

```
2020_express-locallibrary-tutorial
     └────────────┬──────────────┘
       MDN 튜토리얼의 공식 프로젝트 이름
```

`package.json`도 확인해 준다:

```json
{
  "name": "express-locallibrary-tutorial",
  "version": "0.0.0",
  "private": true,
  ...
}
```

#### 원래 튜토리얼이 만드는 것

MDN 튜토리얼의 최종 목표는 **도서관 재고 관리 웹앱**이다:

| 모델 | 내용 |
|---|---|
| `Book` | 책 (제목, 저자, ISBN, 장르) |
| `Author` | 저자 (이름, 생몰년) |
| `Genre` | 장르 |
| `BookInstance` | 개별 실물 책 (대출 상태) |

CRUD 폼, 목록 페이지, 상세 페이지까지 만드는 **꽤 긴 튜토리얼**(파트 8개)이다.

#### 그런데 이 프로젝트는 거기까지 안 갔다

**모델이 하나도 없다.** 3·7단계에서 보겠지만, 이 프로젝트는 **튜토리얼 초반의 뼈대
상태**에서 멈췄고 그나마도 **일부가 사라졌다.**

**제출용 스크린샷 4장**이 남아 있다:

```
201119160이지용 실행화면-1.PNG
201119160이지용 실행화면-2.PNG
201119160이지용 실행화면-3.PNG
201119160이지용 실행화면-4.PNG
```

`201119160`은 학번이다. **학교 과제 제출물**이라는 뜻이고, 스크린샷이 **당시에는
돌아갔다는 유일한 증거**다.
