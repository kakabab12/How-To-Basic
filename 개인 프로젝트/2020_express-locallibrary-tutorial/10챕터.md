# 10챕터 — 실행법과 함정

> **Express 지역 도서관 (2020)** · 10/10챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

#### 실행

```bash
cd "개인 프로젝트/2020_express-locallibrary-tutorial/복구"
npm install
npm start
# http://localhost:3000
```

#### ⚠ 함정 ① — `/`에 접속하면 500 에러가 난다

**7단계의 `Board is not defined`** 때문이다. 서버는 뜨지만 홈페이지가 죽는다.

**고치려면** `routes/index.js`를 이렇게:

```javascript
router.get('/', function(req, res, next) {
  res.render('index', { title: 'Express' });     // Board 제거
});
```

**한 줄 고치면 "Welcome to Express" 페이지가 뜬다.**

#### ⚠ 함정 ② — MongoDB가 필요하다(고 되어 있다)

```javascript
var mongoDB = 'mongodb://127.0.0.1:27017/kababdb';
```

**로컬 MongoDB가 없으면** 콘솔에 연결 에러가 계속 찍힌다.
**다만 서버는 뜬다**(6단계) — 그리고 함정 ①을 고치면 **DB 없이도 페이지가 나온다**
(`Board`를 안 쓰면 DB를 안 건드리니까).

**MongoDB를 띄우려면:**
```bash
# Windows: MongoDB Community Server 설치 후
mongod --dbpath C:\data\db
```

#### ⚠ 함정 ③ — 최신 Node.js에서 안 돌 수 있다

| 문제 | 증상 |
|---|---|
| `mongoose 5.x` | Node 18+ 에서 경고 또는 오류 |
| `pug 2.0.0-beta11` | 최신 Node에서 빌드 실패 가능 |
| `node_modules`가 낡음 | 커밋된 게 2020년 것 |

**`npm install`을 다시 돌리는 게 낫다** — 커밋된 `node_modules`는 6년 전 바이너리라
현재 Node 버전과 안 맞을 수 있다.

**그래도 안 되면** `package.json`의 버전을 올린다:

```json
"mongoose": "^8.0.0",
"express": "^4.21.0",
"pug": "^3.0.0"
```

단 `mongoose 6+`에서는 `useNewUrlParser`/`useUnifiedTopology` 옵션을 **빼야** 한다(6단계).

#### ⚠ 함정 ④ — `serverstart`는 Windows에서 안 된다

```bash
npm run serverstart      # DEBUG=... 는 유닉스 문법
```

Windows에서는:
```powershell
$env:DEBUG="express-locallibrary-tutorial:*"; npm run devstart
```

#### 이 프로젝트를 읽는 이유

**동작하지 않는 프로젝트인데 왜 읽나?**

| 배울 것 | 어디서 |
|---|---|
| **Express의 표준 구조** | `bin/www` ↔ `app.js` 분리 (4단계) |
| **미들웨어 파이프라인** | `app.use()` 순서가 곧 실행 순서 (5단계) |
| **404가 왜 맨 마지막인가** | 5단계 |
| **에러 핸들러 인자 4개** | 5단계 |
| **템플릿 상속** | `extends` / `block` (8단계) |
| **의존성 버전 표기** | `~` vs `^` vs 고정 (9단계) |
| **★ 잃어버린 코드의 흔적을 읽는 법** | 3·7·8단계 |

**마지막 항목이 가장 값지다.** `Board.find()` 한 줄, `kababdb`라는 DB 이름,
`복구`라는 폴더 이름 — **세 조각을 맞추면 사라진 파일의 존재를 증명**할 수 있다.

> **그리고 이건 실무에서 늘 하는 일이다.** 남이 짜다 만 코드, 문서 없는 레거시를 받아
> "여기 원래 뭐가 있었나"를 추론하는 것. 이 폴더는 그 연습 문제다.

#### 이 저장소에서의 위치

**2020년, 가장 오래된 프로젝트**다. 이후 흐름과 비교해 보면:

| | 2020 Express | 2026 젯슨/메디컬 |
|---|---|---|
| 에러 처리 | `err`를 무시 (7단계) | `try/except` 도처에 |
| 스타일 | 콜백 | async / 스레드 |
| 설정 | 하드코딩 | 상수 블록 / `config.yaml` |
| 완성도 | **미완성** | 동작 확인됨 |

**6년치 성장이 저장소 안에 남아 있다.**
