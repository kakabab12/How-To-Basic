**파일 두 개가 전부**다. `pip install`도, 빌드도, 의존성도 없다.
**더블클릭하면 돌아간다** — 그게 배치 스크립트의 최대 장점이다.

#### 299줄의 구성

```
줄        내용
──────────────────────────────────────────
1~6       설정 (chcp, color, title)      ← 3단계
8~15      자동 관리자 권한 획득           ← 2단계  ★
17~19     창 크기 고정                   ← 3단계
21~70     :MENU  — ASCII 아트 + 메뉴 + 분기  ← 4단계
72~114    :UPDATE      (6단계 작업)      ← 5단계
116~166   :CLEAN       (8단계 작업)      ← 6단계
168~229   :KILLPROCESS (게임 모드)       ← 7단계  ★
231~...   :PINGTEST
...       :MYINFO
...       :EXIT
```

**ASCII 아트가 15줄**을 차지한다(27~41줄). "SYSTEM MASTER" 로고를 문자로 그린 것이다.

```
title SYSTEM MASTER TOOL (COMPACT)
```

제목이 **`(COMPACT)`**인 게 힌트다 — 더 큰 버전을 만들려다 줄인 흔적이거나,
창에 맞게 압축했다는 뜻으로 보인다.
