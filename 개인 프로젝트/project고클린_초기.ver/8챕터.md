# 8챕터 — alpha.ver로 넘어가며 — 무엇이 바뀌었나

> **고클린 초기.ver** · 8/8챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

#### 실행

```
project고클린_초기.ver/update.bat  더블클릭
```

UAC 창이 뜬다(2단계). 그다음 메뉴에서 1~6을 입력한다.

#### 두 버전 총정리

| 항목 | 초기.ver (배치) | alpha.ver (Python) |
|---|---|---|
| **줄 수** | 299 | 568 |
| **의존성** | 없음 | `customtkinter` |
| **권한 확인** | `fltmc` 실패 여부 (간접) | `IsUserAnAdmin()` (정식 API) |
| **재실행** | `powershell -Verb RunAs` | `ShellExecuteW("runas")` |
| **한글** | `chcp 65001` (**예방**) | `safe_decode` (**방어**) |
| **UI** | ASCII 아트 + `mode con` 고정 | customtkinter 다크 GUI |
| **진행 표시** | `[1/8]` 텍스트 | `indeterminate` 진행 막대 |
| **동시성** | ❌ 순차 (기다림) | ✅ 스레드 |
| **중복 실행 방지** | 불필요 (한 번에 하나) | 버튼 `disabled` |
| **다국어** | ❌ | ✅ 한/영 실시간 |
| **GPU 드라이버** | ❌ **NVIDIA 하드코딩** | ✅ **제조사 자동 감지** |
| **청소 단계** | **8** (`sfc` 포함) | 5 |
| **게임 모드 확인** | ✅ **Y/N** | ❌ **없음** |
| **배포** | `.bat` | **PyInstaller `.exe`** |

#### 무엇을 얻었나

| 개선 | 왜 가능했나 |
|---|---|
| **GPU 자동 감지** | Python에서 `wmic` 출력 파싱 + 분기가 쉽다 |
| **UI가 안 멈춤** | `threading` |
| **다국어** | 딕셔너리 자료구조 |
| **exe 배포** | PyInstaller |
| **인코딩 방어** | `try/except` 다단 폴백 |

#### 무엇을 잃었나

| 손실 | 이유 |
|---|---|
| **`sfc /scannow`** | GUI에서 30분 작업의 진행률을 못 보여준다 |
| **게임 모드 Y/N 확인** | 옮기면서 빠뜨린 것으로 보임 — **되살리는 게 맞다** |
| **의존성 0** | `customtkinter` 필요 (→ exe로 상쇄) |
| **더블클릭 즉시 실행** | Python 설치 필요 (→ exe로 상쇄) |

#### 이 폴더를 읽는 이유

**"같은 문제를 두 도구로 풀면 어떻게 달라지는가"의 교보재**다.

- 권한 → `fltmc` 트릭 vs `IsUserAnAdmin()` API
- 한글 → `chcp` 예방 vs `safe_decode` 방어
- 진행 → `[1/8]` vs 진행 막대
- 상태 → `goto MENU` 루프 vs 이벤트 루프

**어느 쪽도 일방적으로 우월하지 않다.** 배치는 의존성 0에 더블클릭이면 끝이고,
`sfc` 같은 장시간 작업에 오히려 낫다. Python은 그 대신 GPU 감지·다국어·비동기를 얻었다.
