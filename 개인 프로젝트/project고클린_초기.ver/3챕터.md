# 3챕터 — 터미널 UI 세팅 — `chcp` · `color` · `mode con`

> **고클린 초기.ver** · 3/8챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

```batch
@echo off
title SYSTEM MASTER TOOL (COMPACT)
:: 유니코드(UTF-8) 설정
chcp 65001 > nul
:: 0b = 검은 배경에 하늘색 글씨
color 0b
...
:: 창 크기 설정 (화면에 딱 맞게 축소: 가로 100, 세로 50)
mode con: cols=110 lines=40
cls
```

**네 줄이 각각 다른 문제를 푼다.**

#### `@echo off` — 명령을 화면에 찍지 않는다

배치는 기본적으로 **실행할 명령을 먼저 출력하고 실행**한다. 끄지 않으면 화면이
`C:\...>echo 안녕` 같은 걸로 도배된다. **모든 배치 스크립트의 첫 줄**인 이유다.

#### `chcp 65001` — 한글 깨짐 방지

| 코드페이지 | 뜻 |
|---|---|
| 949 (기본) | CP949 — 한국어 Windows 기본 |
| **65001** | **UTF-8** |

**이 `.bat` 파일 자체가 UTF-8로 저장**되어 있다. CMD가 CP949 모드면 파일 안의 한글을
CP949로 해석해서 **`시스템 업데이트`가 `�쒖뒪�� �낅뜲�댄듃`로 깨진다.**

`> nul`은 `Active code page: 65001` 메시지를 숨긴다.

> **9단계에서 alpha.ver과 비교하면 흥미롭다.**
> 배치: **`chcp 65001`** — "환경을 UTF-8로 바꾸자" (**예방**)
> Python: **`safe_decode`** — "뭐가 오든 받아내자" (**방어**)
> **같은 문제, 정반대 전략**이다.

#### `color 0b` — 미학

```
color 0b
      ││
      │└── b = 하늘색(Aqua) 글씨
      └── 0 = 검은 배경
```

alpha.ver의 `COLOR_ACCENT = "#00E5FF"`(시안), `COLOR_TERMINAL_TXT = "#00FF41"`(매트릭스
그린)으로 이어지는 **"전문가 도구" 미학**의 출발점이다.

#### `mode con: cols=110 lines=40` — ASCII 아트 방어

**가장 실전적인 한 줄**이다.

배치 스크립트는 **창 크기를 모른다.** 사용자가 작은 창에서 열면:

```
정상 (110칸)                    좁은 창 (60칸)
  ,~. -#@ ~#~   !@#@@@: ;$,      ,~. -#@ ~#~   !@#@@@: 
  ,$*,.#@ -#~    ..-$$. ;#-.     ;$,
                                 ,$*,.#@ -#~    ..-$$. 
                                 ;#-.
   ← 로고가 보인다                ← 줄바꿈으로 완전히 무너진다
```

**ASCII 아트는 줄 폭이 고정이어야 한다.** 15줄짜리 로고와 가운데 정렬된 메뉴가 전부
무너지므로 **창 크기를 강제로 고정**했다.

> **주석과 코드가 안 맞는다** — 주석은 `가로 100, 세로 50`인데 코드는 `cols=110 lines=40`.
> 조정하면서 주석을 안 고친 흔적이다. 무해하지만 흔한 일이다.
