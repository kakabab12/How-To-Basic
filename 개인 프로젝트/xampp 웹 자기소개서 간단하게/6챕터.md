# 6챕터 — ★ 이미지가 전부 깨진다

> **XAMPP 웹 자기소개서 (2025)** · 6/8챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

**`Web/` 폴더의 실제 내용:**

```
Web/
├── index.html
├── style1.css
└── style2.css
```

**끝이다. 이미지가 하나도 없다.**

**그런데 `index.html`은 이미지를 5개 참조한다:**

```html
<img src="myphoto.jpg" alt="본인 사진" width="200" />
<img src="myphoto.jpg" alt="친구 사진" width="200" />
<img src="myphoto.jpg" alt="카자루 사진" width="200" />
<img src="naver.png" alt="네이버" width="80">
<img src="google.png" alt="구글" width="80">
```

#### 이미지는 옆 폴더에 있다

```
xampp 웹 자기소개서 간단하게/
├── Web/                    ← index.html 이 여기
│   └── (이미지 없음)
└── 웹 과제/                 ← 이미지가 여기!
    ├── naver.png           ✅ 있다
    ├── google.png          ✅ 있다
    └── myphoto.jpg         ❌ 없다 (이것도 없다!)
```

**`naver.png`, `google.png`는 `웹 과제/`에 있고**, **`myphoto.jpg`는 아예 어디에도 없다.**

#### 지금 열면 이렇게 보인다

```
 안녕하세요! 이지용입니다

[🖼 본인 사진]  [🖼 친구 사진]  [🖼 카자루 사진]      ← 깨진 아이콘 + alt 텍스트

┌──────┬──────────────┐
│ 항목 │ 내용         │
│ 이름 │ 이지용       │      ← 테이블은 정상
└──────┴──────────────┘

저는 개발자가 되고 싶습니다!     ← 스타일 정상

[🖼 네이버]  [🖼 구글]                                ← 깨짐
```

**`alt` 속성 덕분에 뭐가 있어야 할지는 보인다**(3단계에서 잘한 점으로 꼽은 이유).

#### 왜 이렇게 됐나

**제출할 때는 분명 정상이었다** — 스크린샷 4장이 증거다.

추정:
1. 원래 `Web/`에 이미지가 다 있었다
2. 제출 후 정리하면서 이미지를 `웹 과제/`로 옮겼다
3. `myphoto.jpg`는 **개인 사진이라 일부러 뺐을 수 있다**

**`웹 과제/`에 있는 것들:**

```
IMG_5634 사본.jpg
KakaoTalk_20250710_150719225.jpg      ← 이 중 하나가 myphoto.jpg 였을 수 있다
KakaoTalk_20250710_165541020.png
```

#### 고치려면

```bash
cd "xampp 웹 자기소개서 간단하게"
cp "웹 과제/naver.png" "웹 과제/google.png" Web/
# myphoto.jpg 는 아무 이미지나 넣으면 된다
```

#### 같은 사진을 3번 쓴다

```html
<img src="myphoto.jpg" alt="본인 사진" />
<img src="myphoto.jpg" alt="친구 사진" />      ← 같은 파일!
<img src="myphoto.jpg" alt="카자루 사진" />    ← 같은 파일!
```

**alt는 다른데 src는 같다.** 사진 3장을 요구했는데 **1장만 있어서 돌려썼거나**,
급하게 만들면서 src를 안 바꾼 것이다.

> **"카자루"**가 누구인지는 알 수 없다. 친구 이름이거나 반려동물이거나.
