# 4챕터 — `style1.css` — 요소 스타일

> **XAMPP 웹 자기소개서 (2025)** · 4/8챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

**21줄 전체:**

```css
body {
  font-family: Arial, sans-serif;
  padding: 20px;
  background-color: #f8f8f8;
}

.title {
  color: #2c3e50;
  font-size: 2em;
  margin-bottom: 10px;
}

table {
  margin-top: 20px;
  border-collapse: collapse;
  width: 50%;
}

table th, table td {
  padding: 10px;
  text-align: left;
}
```

#### 잘 쓴 것

| 코드 | 왜 좋은가 |
|---|---|
| **`font-family: Arial, sans-serif`** | **폴백 체인** — Arial이 없으면 시스템 sans-serif |
| **`border-collapse: collapse`** | 테이블 이중 테두리 제거 — **없으면 선이 두 겹으로 보인다** |
| **`font-size: 2em`** | 상대 단위 — 부모 크기에 비례 |
| **`width: 50%`** | 상대 폭 — 창 크기에 반응 |
| **`table th, table td`** | 그룹 선택자 — 중복 제거 |

#### `border-collapse: collapse`가 실전적이다

```
없으면 (separate, 기본값)        있으면 (collapse)
┌─────┐┌─────┐                  ┌─────┬─────┐
│  A  ││  B  │                  │  A  │  B  │
└─────┘└─────┘                  ├─────┼─────┤
┌─────┐┌─────┐                  │  C  │  D  │
│  C  ││  D  │                  └─────┴─────┘
└─────┘└─────┘
 셀마다 테두리 → 선이 두 겹      테두리 공유 → 깔끔
```

**`<table border="1">`만 쓰면 선이 두 겹으로 보인다.** 그걸 CSS로 고쳤다.

#### `em` 단위

```css
.title { font-size: 2em; }
```

**`em`은 부모의 폰트 크기 배수**다. 부모가 16px이면 `2em` = 32px.
`px`로 박지 않아서 **사용자가 브라우저 기본 글꼴 크기를 키우면 같이 커진다** — 접근성에 좋다.

#### 한글 폰트가 없다

```css
font-family: Arial, sans-serif;
```

**Arial에는 한글이 없다.** 이 페이지는 전부 한글인데.

**결과**: 브라우저가 **한글만 시스템 기본 폰트로 폴백**한다(맑은 고딕 등).
**영문과 한글의 폰트가 달라진다** — 의도한 건 아닐 것이다.

**고치려면:**
```css
font-family: Arial, "맑은 고딕", "Malgun Gothic", sans-serif;
```

> **동작에는 문제없다.** 브라우저의 폰트 폴백이 알아서 처리한다.
