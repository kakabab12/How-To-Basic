> **XAMPP 웹 자기소개서 (2025)** · 8/8챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

#### 실행 — 3가지 방법

**방법 1: 그냥 브라우저로 열기 (가장 쉽다)**

```
Web/index.html 을 더블클릭
```

**정적 파일이라 이걸로 충분하다.** PHP도 DB도 안 쓴다.

**방법 2: XAMPP (과제 원래 방식)**

```
1. XAMPP 설치
2. Web/ 폴더를 C:\xampp\htdocs\ 에 복사
3. XAMPP Control Panel → Apache [Start]
4. http://localhost/Web/
```

**방법 3: 파이썬 내장 서버 (XAMPP 없이)**

```bash
cd "개인 프로젝트/xampp 웹 자기소개서 간단하게/Web"
py -m http.server 8000
# http://localhost:8000
```

**XAMPP를 설치할 필요 없이 로컬 서버 경험을 할 수 있다.**

#### ⚠ 알고 열 것

| # | 문제 | 결과 |
|---|---|---|
| 1 | **이미지 5개 없음** (6단계) | 깨진 아이콘 + alt 텍스트 |
| 2 | **YouTube `watch` URL** (7단계) | 빈 회색 박스 |
| 3 | 한글 폰트 미지정 (4단계) | 영문/한글 폰트가 다름 |

**고치려면:**

```bash
# 1. 이미지 복사
cp "웹 과제/naver.png" "웹 과제/google.png" Web/
# myphoto.jpg 는 아무 이미지나

# 2. index.html 의 iframe src 를
#    https://www.youtube.com/watch?v=9XttLI0oH0I...  →
#    https://www.youtube.com/embed/9XttLI0oH0I
```

#### 이 과제에서 배울 점

| 배울 것 | 어디서 |
|---|---|
| **요구사항 번호를 코드에 남긴다** | 2단계 ★ |
| **`alt`는 깨졌을 때 진가를 발휘한다** | 3·6단계 |
| **`<i>` vs `<em>`의 차이** | 3단계 |
| **`border-collapse`가 왜 필요한가** | 4단계 |
| **폰트 폴백 체인** | 4단계 |
| **유틸리티 클래스 = Tailwind의 원리** | 5단계 ★ |
| **`watch` URL ≠ `embed` URL** | 7단계 ★ |
| **상대 경로는 파일을 옮기면 깨진다** | 6단계 ★ |

#### 가장 값진 교훈

**"제출할 때 돌아갔다"가 "지금도 돌아간다"를 뜻하지 않는다.**

스크린샷 4장은 2025-07-10에 정상 동작했음을 증명한다. 그런데 **지금 열면 깨진다** —
파일을 옮겼기 때문이다(6단계).

> **이게 상대 경로의 본질적 위험**이다. `src="naver.png"`는 **"내 옆에 있는 naver.png"**를
> 뜻한다. HTML을 옮기거나 이미지를 옮기면 **조용히 깨진다.** 에러도 안 난다.
>
> 같은 문제를 [광명테크 프로젝트](../../광명테크/README.md)는
> `config_loader.py`로 풀었고, [ASP 프로젝트](../2024년%202학기ASP_DB연동/README.md)는
> `server.mappath()`로 풀었다. **"상대 경로를 절대 경로로 고정하기"**는
> 언어와 시대를 가리지 않는 문제다.

#### 110줄에서 이만큼 나왔다

**이 저장소에서 가장 작은 코드**인데:
- 잘한 것 6가지 (3·4단계)
- 우연히 맞은 것 2가지 (`<i>`, PRG 아닌 유틸리티 분리)
- 깨진 것 3가지 (6·7단계)
- 배울 개념 8가지

**코드 크기와 읽을 거리는 비례하지 않는다.**
