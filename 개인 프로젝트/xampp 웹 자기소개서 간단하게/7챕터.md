# 7챕터 — ★ YouTube가 안 나온다

> **XAMPP 웹 자기소개서 (2025)** · 7/8챕터 · [📚 목차](README.md) · [개인 프로젝트 전체](../README.md)

---

**추가점수 항목이 실제로는 동작하지 않는다.**

```html
<!-- 추가점수: 좋아하는 유튜브 영상 삽입 -->
<h3> 좋아하는 유튜브 영상</h3>
<iframe width="560" height="315"
  src="https://www.youtube.com/watch?v=9XttLI0oH0I&list=RD9XttLI0oH0I&start_radio=1&ab_channel=RESCENE"
  title="YouTube video player" frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen></iframe>
```

#### 문제 — `watch?v=` 는 iframe에 못 넣는다

```
❌ https://www.youtube.com/watch?v=9XttLI0oH0I      ← 브라우저 주소창용
✅ https://www.youtube.com/embed/9XttLI0oH0I        ← iframe 삽입용
```

**YouTube는 `watch` 페이지의 iframe 삽입을 막는다.** HTTP 응답 헤더로:

```
X-Frame-Options: SAMEORIGIN
```

**"내 도메인에서만 프레임에 넣을 수 있다"**는 뜻이다. `localhost`에서 넣으면 브라우저가
거부한다.

#### 실제로 이렇게 보인다

```
 좋아하는 유튜브 영상
┌──────────────────────────────────┐
│                                  │
│   www.youtube.com에서 연결을      │   ← 빈 회색 박스 + 거부 메시지
│   거부했습니다.                    │
│                                  │
└──────────────────────────────────┘
```

콘솔에는:
```
Refused to display 'https://www.youtube.com/' in a frame because 
it set 'X-Frame-Options' to 'sameorigin'.
```

#### 왜 이런 실수가 나오나

**주소창에서 URL을 그대로 복사했기 때문**이다.

```
YouTube에서 영상 보는 중
    ↓  주소창 복사
https://www.youtube.com/watch?v=9XttLI0oH0I&list=RD...&start_radio=1&ab_channel=RESCENE
    ↓  iframe src 에 붙여넣기
❌
```

**올바른 방법**은 YouTube의 **[공유] → [퍼가기]** 버튼을 누르는 것이다.
그러면 YouTube가 **완성된 `<iframe>` 코드**를 준다:

```html
<iframe width="560" height="315" src="https://www.youtube.com/embed/9XttLI0oH0I"
  title="YouTube video player" frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen></iframe>
```

#### 증거 — 나머지 속성은 완벽하다

```html
title="YouTube video player" frameborder="0"
allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
allowfullscreen
```

**이 속성들은 YouTube 퍼가기 코드를 그대로 복사한 것**이다. 사람이 손으로 쓸 수 있는
문자열이 아니다.

**즉: 퍼가기 코드를 가져왔는데, `src`만 주소창 URL로 바꿔 끼웠다.**
아마 "내가 보던 그 재생목록으로 연결하고 싶어서"였을 것이다 —
`&list=RD...&start_radio=1`이 **라디오 재생목록** 파라미터다.

#### 고치려면

```html
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/9XttLI0oH0I"
  title="YouTube video player" frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen></iframe>
```

**재생목록까지 살리려면:**
```html
src="https://www.youtube.com/embed/9XttLI0oH0I?list=RD9XttLI0oH0I"
```

> **`ab_channel=RESCENE`** — RESCENE은 K-pop 그룹이다. 취향이 남았다.

> **그런데 15점 만점을 받았다.** 추가점수 항목이 안 나오는데도.
> 채점자가 안 봤거나, 기본 8개 항목만으로 만점이었거나,
> **제출 당시 스크린샷**(`화면 캡처 2025-07-10 170536.png`)에는 나왔을 수도 있다 —
> 다만 `X-Frame-Options`는 2013년부터 적용된 정책이라 **당시에도 안 나왔을 가능성이 높다.**
