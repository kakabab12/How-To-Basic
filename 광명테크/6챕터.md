# 6챕터 — 2층 전처리 — 거울 반전

> **광명테크 제스처 인식** · 6/15챕터 · [📚 목차](README.md)

---

`src/inference/preprocessor.py` — 프로젝트에서 가장 짧은 파일이다.

```python
class Preprocessor:
    def __init__(self, config):
        self._is_mirror = config["camera"]["mirror"]

    def preprocess_frame(self, frame):
        """frame(BGR) -> input_tensor. 거울 모드면 좌우 반전한다."""
        if self._is_mirror:
            frame = cv2.flip(frame, 1)
        return frame
```

**한 줄짜리 기능에 왜 클래스를 만들었나?** 나중에 리사이즈·정규화·ROI 크롭이 붙을 자리를
미리 잡아 둔 것이다. 지금은 거울 반전만 한다.

**이 한 줄이 뒤에서 두 번 사고를 친다.** `mirror=true`면 화면에 보이는 좌우가 뒤집히므로:

- **7단계**: MediaPipe의 handedness 라벨(왼손/오른손)이 사용자 실제 좌우와 어긋난다
- **9·10단계**: RTMPose의 "왼손목"이 사용자의 실제 오른손이 된다

두 모델이 각자 다른 방식으로 좌우를 말하기 때문에 **보정도 각자 따로** 해야 한다.
그래서 `flip_handedness`(MediaPipe용)와 `person_lock`의 거울 보정(RTMPose용)이 **따로** 있다.
